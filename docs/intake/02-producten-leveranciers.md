## Instructie per kolom

| Kolom | Uitleg | Voorbeeld |
|---|---|---|
| **Productnaam** | Naam zoals op de factuur | Friet diepvries 9mm |
| **Eenheid** | Hoe meet je het product | kg / stuk / liter / doos |
| **Verpakkingsgrootte** | Hoeveel zit er in één besteleenheid | 10 kg per zak |
| **Houdbaarheid** | Hoe lang blijft het goed | Laag = weken/maanden / Midden = dagen / Hoog = <1 dag |
| **Leverancier** | Naam van de leverancier | Hanos |
| **Soort leverancier** | Groothandel of vers/lokaal | Groothandel / Vers |
| **Minimale voorraad** | Wat wil je altijd minimaal op voorraad hebben | 10 kg |
| **Verbruik per gast** | Hoeveel gebruik je gemiddeld per couvert | 0.2 kg per gast |
| **Besteleenheid afronden?** | Mag je een halve verpakking bestellen, of altijd hele dozen? | Ja = altijd hele dozen |
